# Contributing to AI News Digest

We love your input! We want to make contributing to this project as easy and transparent as possible.

## How to Contribute

1.  **Fork the repo** and create your branch from `main`.
2.  **Install dependencies**: `pip install -r requirements.txt`.
3.  **Test your changes**: Run `python discord_fetch.py` (or relevant script) to ensure nothing breaks.
4.  **Submit a Pull Request**: Describe your changes clearly.

## Reporting Bugs

- Ensure the bug was not already reported.
- Open a GitHub Issue with a clear title and description.
- Include logs or reproduction steps.

## Coding Style

- Follow PEP 8 guidelines.
- Keep functions small and focused.
- Add comments for complex logic (especially in `curate.py` or regex handling).
